recently discord switched to kiply for GIFs which is pretty terrible

this plugin adds a button for giphy search. when you click the button, it will pop up a search window. click on a GIF it will copy the URL to the clipboard and you can paste it into chat. 

you will need a giphy API key for this to work, you can get one for free at https://developers.giphy.com/

then just paste it into the code

I made this with gemini on a whim so don't @ me about crappy code, I have no idea what I'm doing
